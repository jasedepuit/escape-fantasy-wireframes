AI Suggestion Overlay
A bottom sheet modal triggered by tapping "Get AI Suggestion" in the Rider Selection screen. Surfaces a recommended rider for the specific stage, with transparent reasoning so the user can decide whether to trust the suggestion or browse manually.
Entry Point
1
"Get AI Suggestion" banner — visible in the Rider Selection screen above the rider list. Tapping it triggers this modal. Distinct from Auto Pick (My Picks screen), which silently fills all remaining stages without showing this overlay.
Suggested Rider
2
Stage context — modal header shows the stage number and key characteristic (e.g. "Mountain finish · Col de la Croix de Fer") so the user understands the basis for the suggestion.
3
Recommended rider card — highlighted with an orange border to distinguish it from a regular rider row. Shows photo, name, team, speciality, and their rank among the user's available riders for this stage.
Reasoning
4
Stage type fit — how well the rider's profile matches the stage characteristics. Key metric shown in orange.
5
Current form — recent race results summarised in plain language. Helps the user sanity-check the suggestion against what they may already know.
6
vs. alternatives — brief head-to-head note comparing the suggestion against the user's other available riders. Reduces decision anxiety without requiring the user to manually compare.
7
Historical performance — past results at this specific finish or race. Particularly useful for classics and well-established stage finishes.
8
Other suggestions — tapping this row expands to show the #2 and #3 ranked alternatives with their reasoning, in case the user wants to compare before deciding.
Actions
9
Pick CTA — confirms the suggested rider for this stage and dismisses the modal. Same contextual label pattern as the Rider Profile: "Pick [Name] for Stage [N]".
10
Browse manually — dismisses the modal and returns the user to the full rider list, with no pick made. Preserves user control.
This overlay is stage-specific. Auto Pick (triggered from My Picks) is a separate, silent bulk action that fills all remaining unpicked stages and does not show this screen.
