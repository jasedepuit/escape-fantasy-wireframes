Screen 17 · Notification Preferences — Annotations
① OS Permission Banner
Shown only when the user has not yet granted (or has revoked) iOS/Android notification permissions. "Open Settings" deep-links to the device's notification settings for the app. The banner disappears once permission is granted. Toggles below are still visible and configurable regardless — preferences are saved for when permission is eventually granted.
② Fantasy Section
Covers competition-level events: pick lock reminders, personal leaderboard position changes, and league leaderboard changes. All three default on — high-value signals the user is unlikely to want to miss. "Picks closing soon" replaces the old "Stage Picks closing soon" label with a shorter, clearer name.
③ League Section
League enrolled in competition — default on. Tells members when their director has entered the league into a new competition so they don't miss the picks window.

New member joined — default off. Directed at league directors. Most members won't care; noisy for large public leagues.
④ Chat Section
Chat notifications are per-app, not per-league — a single toggle covers all leagues the user is in. If chat volume becomes a concern, a per-league granularity could be added in a future iteration (accessible from League Settings).
⑤ Mentions Only
Subordinate toggle — only meaningful when New chat message is on. When enabled, suppresses all chat notifications except direct @mentions, reducing noise for active leagues. Default off (users receive all messages by default). Consider disabling this toggle visually when its parent is off.
