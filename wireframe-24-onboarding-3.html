Screen 24 · Onboarding 3 — Annotations
① Competition Picker
Shows the same competitions available in the full Competition Browser (Screen 02), filtered to currently active or upcoming only — past/completed events are excluded. This is a curated, frictionless first-entry moment. Joining here is equivalent to joining from the Competition Browser — same logic applies.
② Live Competition — Already Joined
If a live competition exists, it is shown first and can be pre-selected (or even auto-joined) so the user can immediately make picks. The Joined ✓ state and card selection reflect this. For mid-race joining, the user can still score from the next available stage — they don't miss out entirely.
③ "Go to Home" CTA
Final button in the onboarding flow. Navigates to the Home screen (Screen 01). If the user has joined a competition, the Home screen immediately shows that competition card. The onboarding flag is set here — this screen never appears again on subsequent launches.
Browse All Competitions
Secondary CTA. Also navigates to Home but opens the Competition Browser tab. Equivalent to the user navigating there manually — provided for users who don't see a relevant competition in this curated list.
Format Badges
Each competition shows which format(s) it supports — Stage Race (orange), Fantasy Teams (blue) — using the same colour language as the rest of the app. Useful context before committing to an entry, especially for new users who just learned the distinction on Screen 23.
