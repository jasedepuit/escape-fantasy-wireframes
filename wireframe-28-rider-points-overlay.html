Wireframe 28 — Rider Points Breakdown Overlay
① Trigger
Tapping the points value next to any rider in the Team Overview (Wireframe 20) opens this overlay. The same action is available when viewing another team's roster from the Team Leaderboard (Wireframe 27). The tapped value is underlined to signal it is interactive.
② Rider header
Avatar uses the rider's class colour as background with their initials. Shows rider name, pro team, and class pill. If a headshot photo is available from the data feed it should replace the initials avatar.
③ Total points
Cumulative points across all completed stages for this competition. Shown prominently in orange — matches the value the user tapped to open this overlay.
④ Per-stage breakdown
Each row shows the stage, the rider's finish position, and the points earned. The Pts column shows a compact breakdown by source (stage finish / GC classification / Sprint or KOM classification). The Total column is the sum for that stage. Stages with a standout result can be highlighted (e.g. green for podium). DNF stages show 0 pts and a dimmed row.
Dismissal
Drag down or tap the dimmed area behind the sheet to dismiss. No explicit close button needed.
