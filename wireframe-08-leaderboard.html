Screen 8
Leaderboard
Accessed from the "Leaderboard ›" button on active competition tiles in the Home screen and Competition Browser. Shows all players ranked by points for the selected competition, with the logged-in user's row always highlighted.
Header & Tabs
1
Competition name — identifies which competition's leaderboard is being viewed. Back arrow returns to wherever the user came from (Home or Competition Browser).
2
Search icon — find a specific player by name within the leaderboard. Useful in large competitions where the user wants to find a friend.
3
Leaderboard | Leagues tabs — Leaderboard shows the global competition ranking. Leagues tab filters the same list to only show members of a specific league. If the user isn't in any leagues for this competition, the Leagues tab shows an empty state with an invite prompt.
My Stats
4
Pinned summary card — always visible above the list regardless of scroll position. Shows the user's current rank (in orange), total points, and points behind the leader. Gives instant context without needing to scroll to find your own row.
Player Rows
5
Top 3 rank highlight — positions 1–3 show the rank number in orange to visually distinguish the podium from the rest of the field.
6
Position change badge — green upward arrow with number of places gained, red downward arrow with places dropped. Only shown when there has been a change since the previous stage result. No badge = no change.
7
Logged-in user row — highlighted with an orange tint and bold name/points so the user can quickly locate themselves in the list when scrolling. Matches the rank shown in the My Stats card above.
8
Tap any row — navigates to the Per-Stage Results screen (Screen 7) for that player, showing their pick history across all stages.
The Leagues tab shares this screen's layout but filters results to league members only. The screen map shows this as a separate planned screen (League Detail — Standings) but the tab pattern means no full navigation change is required.
