Screen 11
Create League
Simplified single-form design — the Standard/Advanced tab split has been removed. Usage data from TdF 2025 (541 leagues) drove the decisions below.
What's in the form
1
Name, Logo, Colour — kept. Logo used by 58% of leagues, custom colour by 45%. The colour picker moves from a freeform hex input to 8 preset swatches, keeping cards visually consistent across the app.
2
Include in Competitions — new field. Multi-select from competitions the user has joined. Replaces the implicit assumption that a league belongs to one race, enabling season-long leagues across multiple events.
3
Invite Only — kept. 91% of leagues are private, so it defaults on. Turning it off makes the league discoverable on the All Leagues tab without needing a code.
4
Send Daily Digest Email — kept but defaults off. 30% usage is meaningful enough to surface here. "Use BCC" is removed — it's a detail directors can manage in league settings after creation.
What was removed and why
5
Ranking by Time — 3% (16/541 leagues). Removed from creation. By Points is the overwhelming default. Time-based ranking can be reinstated as a hidden setting if truly needed.
6
Cap Time Loss Per Stage — 93% on. This is default behaviour now, not a choice. Surfacing it as a toggle implied it was reasonable to turn off, which the data shows almost nobody does.
7
Allow Different Pick (4%) and Second Chance Pick (5%) — too niche for the creation flow. Both are available in league settings post-creation for directors who specifically need them.
8
Standard / Advanced tab split — with the above removals, the Advanced tab shrank to nothing. A single-page form is clearer and faster.
After tapping Create League, the user is taken to a Share Invite sheet (native OS share sheet with invite code + link). The new league appears at the top of My Leagues. Directors can access the removed low-usage settings via the gear icon in League Detail.
