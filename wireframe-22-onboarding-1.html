Screen 22 · Onboarding 1 — Annotations
① Skip Button
Available on all three onboarding screens. Tapping jumps directly to the Home screen (Screen 01). A first-time-seen flag is set in local storage regardless of whether the user skips, so the onboarding never shows again. The flag can be reset via Account → App Settings for testing.
② Escape Fantasy Branding
Uses the Escape Fantasy stacked logo (not the EC master logo) as this is the product's own identity moment. White variant in dark mode, black in light mode — same switching pattern as the app header logo.
③ Feature Cards
Three concise value propositions. Not interactive — purely informational. Copy should be short enough to scan in one glance. Icons use class colours from the rest of the app (orange, blue, green) to start building visual language familiarity.
Progress Dots
Three dots indicating position in the 3-screen onboarding flow. Active dot is wider (pill shape) in orange. Tapping dots does not navigate — only Next / Back buttons do, keeping the interaction unambiguous.
