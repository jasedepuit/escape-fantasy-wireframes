Screen 16 · Account & Settings — Annotations
① Profile Header
Replaces the plain list row from the old design with a prominent profile card showing avatar initials, display name, email, and EC membership badge. Tapping navigates to a profile edit screen (not yet in scope). The EC Member badge confirms active Escape Collective membership — relevant since access to content on the news tab may depend on subscription status.
② Theme Toggle
Inline Light / Dark segmented control — no chevron, no sub-screen. Applies immediately. Default follows the device system setting on first launch. Choosing here overrides the system preference and persists across sessions.
③ Notifications Row
Navigates to the Notification Preferences screen (Screen 17). Grouped here under "App" alongside Announcements and Fantasy Rules for logical cohesion.
④ Log Out
Destructive action — label and icon shown in red to signal consequence. No chevron (not a sub-screen). Should trigger a confirmation sheet ("Are you sure?") before logging the user out. Tapping away from the sheet cancels.
⑤ Version / Footer
App version string pulled from build metadata. Terms and Privacy open in the default browser (same rationale as news articles). Kept in footer to avoid cluttering the settings list.
