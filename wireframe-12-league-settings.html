Screen 12
League Settings
Reached via the gear icon in the League Detail header (Screen 10). This screen covers both editable creation fields and the post-creation gameplay settings removed from the creation flow.
Identity & Access
1
Name, Logo, Colour, Competitions — all editable post-creation. Same fields as Create League. Colour uses the 8-swatch picker, not a freeform input.
2
Invite Only + invite code — the current code (e.g. TDF26) is always visible so the director can easily share it. Tapping "Share" opens the native OS share sheet. If Invite Only is turned off, the code row hides and the league becomes discoverable.
Gameplay (Director only)
3
Picks Visible Before Lock — surfaced here rather than at creation. Some leagues prefer transparency, others strategic secrecy. Off by default.
4
Enable Chat — on by default. Directors can disable it for leagues that don't use it, keeping the League Detail tab bar clean.
Save Behaviour
5
Save button in the header is always active (directors land here ready to change things). On tap it confirms changes and returns to League Detail. Toggle and swatch changes update immediately in the UI without requiring Save — Save only persists to the server.
Danger Zone
6
Leave League — available to all members. Tapping shows a confirmation sheet. If the leaver is the director, they're prompted to transfer directorship first.
7
Delete League — director only. Shows a destructive confirmation with the league name typed to confirm. Not shown to regular members.
Entry point: gear icon (⚙) in Screen 10 League Detail header. Non-directors see a read-only version of this screen — Gameplay section and Delete League are hidden, but they can still leave the league.
