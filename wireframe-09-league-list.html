Screen 9
League List
The Leagues tab root. Shows all competitions leagues the user has joined or created, across all active competitions. Entry point to League Detail (Standings + Chat).
Header Actions
1
Search icon — search for a specific league by name. Useful when a user is in many leagues and wants to find one quickly.
2
Create League — opens the Create League flow (Screen 11). Fills the league name, sets competition scope, and generates an invite code.
3
Join by Code — opens the Join by Code screen (Screen 12). User enters a 6-character code shared by a league director.
League Cards
4
Coloured stripe — each league gets a unique accent colour (set by the director or auto-assigned). The stripe and avatar background share the same hue for visual coherence.
5
Competition context — leagues are scoped to a competition. The sub-label shows which race this league belongs to, so users can distinguish leagues across different events.
6
Rank badge + stats row — shows Your Rank (with gold/silver/bronze colouring for top 3), Members count, and Your Pts within this league. Points match the competition score — leagues are just filtered views of the main leaderboard.
7
Unread chat badge — orange pill in the top-right of the card shows the number of unread messages in the league chat. Also reflected as a dot on the Leagues nav icon when any league has unread activity.
8
League Director tag — shown on leagues where the user is the director/admin. Directors can rename the league, remove members, and generate new invite codes.
Other Elements
9
Invite a friend row — persistent CTA at the bottom. Generates a generic invite link that the friend can use to join any of the user's leagues (they choose which on the next screen).
Tapping a league card navigates to League Detail, which has two tabs: Standings (the filtered leaderboard) and Chat. These are planned as Screen 10. The Create League and Join by Code flows are Screens 11 and 12.
