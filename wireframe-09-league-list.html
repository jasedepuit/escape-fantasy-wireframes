Screen 9
League List
The Leagues tab root. Two tabs: My Leagues (leagues the user has joined) and All Leagues (public leagues open to join). League cards show membership and activity stats — not competition-scoped rank or points, since a league can span multiple competitions simultaneously.
My Leagues Tab
1
Create / Join by Code buttons — Create starts the league creation flow (Screen 11: name, logo, colour, settings). Join by Code opens a simple entry screen (Screen 12) for a 6-character invite code.
2
League card stats: Members · Active Comps · Chat — these are intentionally competition-agnostic. A league is a persistent group; rank and points only make sense inside a specific competition's context, accessed from League Detail. Active Comps names the current races in orange when count > 1.
3
Unread chat badge — orange pill on the card and mirrored as a dot on the nav icon. Disappears after the user opens the Chat tab in League Detail.
4
League Director tag — shown on leagues the user created or was granted director rights. Directors can rename the league, remove members, and generate new invite codes (from within League Settings).
All Leagues Tab
5
Public leagues only — leagues created with "Invite Only" turned off appear here. Invite-only leagues are hidden from this list; the only way to join them is via a shared code.
6
Competition filter chips — narrows the list to leagues active in a particular race. "All" shows every public league regardless of competition. Useful when a user wants to join a TdF-specific group.
7
Join button — one tap to join (no confirmation needed for public leagues). Already-joined leagues show a greyed "Joined" state instead. After joining, the league immediately appears in the My Leagues tab.
Tapping a league card on either tab navigates to League Detail (Screen 10), which has Standings and Chat sub-tabs. Rank/points within a specific competition are shown there, not here.
