Screen 10
League Detail
Accessed by tapping any league card on Screen 9. Contains two tabs — Standings (competition-filtered leaderboard scoped to league members) and Chat (message thread with pick-sharing).
Header
1
League name + member count — sub-line also shows "League Director" if the user has that role. Back arrow returns to the League List.
2
Settings gear — visible to all members but reveals different options by role. Directors can rename, change colour, manage members, and regenerate the invite code. Non-directors can only leave the league.
Standings Tab
3
Competition selector — since a league can be active across multiple races, a tap here opens a sheet to switch between them (e.g. Tour de France / Vuelta). Only competitions where ≥2 league members have joined are shown.
4
My position strip — pinned above the list. Shows the user's rank within the league, their points for the selected competition, and points ahead of the next player (or behind, if not in 1st).
5
League standings list — same row format as the main leaderboard (Screen 8) but filtered to league members only. Rank reflects position within the league, not the global competition. Tapping a row navigates to that player's Per-Stage Results (Screen 7).
Chat Tab
6
Message thread — simple group chat scoped to the league. Messages are grouped by sender with a sender label above each group. The user's own messages appear right-aligned with an orange tint bubble.
7
Pick-share cards — tapping the share icon in the input bar lets a user share their pick for a stage as a card in the chat. Other members can tap the card to see the full Rider Profile. This is the primary social mechanic — comparing picks and debating choices.
8
Emoji reactions — tap any message or pick-share to react. Reactions are aggregated and shown as a strip below the message. The user's own reaction is highlighted with an orange tint.
9
Unread divider — on first open after new messages, a divider marks where unread messages begin. The orange dot on the Chat tab disappears once the user scrolls past the divider.
The gear icon opens a settings sheet (not a separate screen). For directors this includes: rename league, change colour, remove member, copy invite code, delete league. For members: leave league only.
