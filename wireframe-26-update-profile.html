Screen 26 — Update Profile
① Personal Details
All fields are editable inline. Tapping a field opens the native keyboard (or a picker for Country). Fields should pre-populate from the user's existing profile data.
② Email Address — read-only
Email is sourced from the user's Ghost/EC membership and cannot be changed here. The field is displayed as read-only with a hint explaining why. Changing email requires going through account management on escapecollective.com.
③ Display Name
The name shown to other users in leagues and on leaderboards. Distinct from First/Last Name. Defaults to First Name on account creation.
④ Picks Reminder Email
Sends a reminder email before the pick cut-off for each stage. Default off. Frequency matches stage cadence for active competitions the user is enrolled in.
⑤ Fantasy Updates Email
General product and competition news emails. Default off.
⑥ Auto Create Leagues
When enabled, the system automatically creates a personal league for the user when a new competition opens. Default off. Useful for users who mostly play solo or invite friends each race.
⑦ Delete Profile — Danger Zone
Permanently deletes the user's Fantasy profile and all associated data (picks, leagues, results). Does not cancel their EC membership — that is managed separately on escapecollective.com. A confirmation dialog should be shown before executing the delete.
